## 📌 Project Overview

This project demonstrates my ability to bridge **financial modeling** and **data analytics** — transforming structured financial data into a fully automated, publication-ready PDF report using R and RMarkdown.

The model integrates the three core financial statements across a **2024–2028 forecast horizon**:

- 🟦 **Income Statement** — Revenue, COGS, operating and net income trends
- 🟩 **Balance Sheet** — Asset growth, capital structure, and liquidity ratios
- 🟥 **Cash Flow Statement** — Operating, investing, and financing activity analysis

Rather than relying on static spreadsheets, every table, chart, and insight in this report is **generated programmatically** — making the analysis fully reproducible and easy to update as new data comes in.

## 📊 Report Highlights

**Income Statement** — Revenue grows from ~$198M to ~$248M over five years. Net income nearly doubles, driven by a declining COGS ratio (43% → 38%) and disciplined SG&A control.

**Balance Sheet** — Total assets more than double from $82M to $177M. Current assets rise from 40% to nearly 70% of total assets, reflecting strong liquidity accumulation.

**Cash Flow** — Operating cash flow is the primary liquidity driver throughout the period. Free Cash Flow expands consistently, reducing reliance on external financing by 2027–2028.
